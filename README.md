# Fantist.Universe.Events
**The Central Data Repository and Interactive Hub for the Fantist SMP**

## Project Overview
`Fantist.Universe.Events` is a React-based web application designed to document, archive, and visualize the evolving narrative of the **Fantist SMP**. This repository serves as the "Universe Nexus"—a centralized terminal for tracking player-driven events, faction conflicts, and mythical archeology within the server.

---

## 🏛️ Active Universe Categories
The current version of the application tracks the following core project streams:

* **falqz (Incognito Infiltration)**
    * **Focus:** Militaristic faction dynamics and infiltration strategies.
    * **Key Entities:** CrimsonzFlame and the "Incognito Valley" fortification.
    * **Objectives:** Intelligence gathering and survival analytics.

* **SmilyCrit (The Sepherian Quest)**
    * **Focus:** Exploration and discovery of the Sepherian Golden Kingdom.
    * **Milestones:** Mastery of the Minecraft Spear and documentation of the 1,000,000-block journey.

* **ReviTech (Technical Operations)**
    * **Focus:** Resource management and end-game dimension control.
    * **Strategic Assets:** Potion/Gap trading systems and End dimension colonization.

* **IcyJK (System Maintenance)**
    * **Status:** [OFFLINE / UNDER MAINTENANCE]
    * **Note:** This module is currently restricted and undergoing internal updates.

## 💻 Technical Specification
* **Front-End:** React 18 with functional component architecture.
* **Styling Engine:** Tailwind CSS for high-fidelity terminal aesthetics.
* **Animation:** Framer Motion for synchronized state transitions.
* **Version Control:** Git-managed repository hosted on GitHub.
